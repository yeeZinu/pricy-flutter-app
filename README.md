## Requirements
### 1. Create `~/.env`

```bash
$ cd ~/pricy-flutter-app | touch .env
```
### 2. Insert the KAMIS certificate keys

```bash
#must be type to string
key = "your key" #required
id = "your id" #required
```
### 3. Run & Debug

```bash
$ pwd | cd ~/pricy-flutter-app
$ flutter run
```
